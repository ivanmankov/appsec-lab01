## Цель
Освоить базовый workflow Git: коммиты с GPG-подписью, ветки, PR, merge, rebase.

## Структура
- `typersteel.py` — CLI на `typer` (аргументы: name, --lastname, --formal/-f)
- `README.md` — описание

## Запуск
```bash
pip install typer
python typersteel.py Иван --lastname Маньков --formal
```
